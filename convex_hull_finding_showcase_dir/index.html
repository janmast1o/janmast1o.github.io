Convex Hull Finding App Showcase

Below is a video showcasing usage of the app, which uses Graham algorithm to find convex hulls for sets of points.

The app uses Qt5 and QCustomPlot as the only external libraries.

App features include:

  • adding points to the canvas by entering them in text format
  • running the algorithm and displaying the final results
  • running the algorithm with visualizng subsequent steps of the Graham algorithm
  • an ability to stop the visualzation
  • an ability to skip to the end in the visualization mode
  • an ability to display current stack of the Graham algorithm
