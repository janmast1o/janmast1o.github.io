Noble Duel Engine Showcase

Below is a video showcasing some capabilites of the hand-built engine, including all movement related ones.

Engine features showcased in the video include:

  • gravity
  • movement and object collisions
  • momentum conservation (objects are able to move each other)
  • slope traversal
  • physics of airborne objects
  • decorative objects (flowers)
  • simple floating platforms

Engine features that have not been shocased in the video:

  • items that can be picked up and dropped
  • item catagories which include simple firearms and thrusting weapons
  • elevators

Also, please ignore that one floating flower, it has been assigned an invalid position on the canvas.
